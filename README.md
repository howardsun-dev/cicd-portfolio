🚀 Setup & Run Locally
```
# Clone the repo
git clone https://github.com/howardsun-dev/cicd-portfolio.git

# Navigate to the project folder
cd cicd-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open http://localhost:5173/
 to view it in your browser.
